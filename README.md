## Descripción
Este repositorio contiene la documentación y planeación inicial para el desarrollo del SGAB, un sistema web diseñado para administrar información académica.

## Objetivo del repositorio
Documentar las fases de análisis, estimación, riesgos y calidad antes de iniciar la programación, demostrando criterio ingenieril.
