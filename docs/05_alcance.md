# Definición del Alcance

## Funcionalidades Incluidas (IN)
- Login seguro para 3 roles (Admin, Profe, Alumno).
- Alta, Baja y Modificación de Alumnos.
- Asignación de materias a grupos.
- Captura de calificaciones parciales y finales.
- Consulta de kárdex por parte del alumno.

## Funcionalidades Excluidas (OUT)
- Pagos de colegiaturas en línea.
- App móvil nativa (Android/iOS).
- Integración con checador biométrico de asistencia.
- Chat en tiempo real entre profesor y alumno.

## Ejemplo de Scope Creep
**Situación:** A mitad del proyecto, el cliente pide agregar un módulo de "Control de Inventario de Cafetería".
**Impacto:** Esto retrasaría la entrega 2 semanas y desviaría el objetivo principal (académico). Se debe rechazar o gestionar como una fase 2.
